Kebijakan Privasi

Berlaku untuk aplikasi Model Uploader (OAuth App Roblox)

Model Uploader adalah tool pribadi untuk mengunggah aset (Model, Animation, Audio) dan mem-publish Place ke Roblox lewat Open Cloud API, menggunakan fitur Login dengan Roblox (OAuth2) sebagai satu-satunya metode otentikasi.

Data yang kami akses

Saat kamu login lewat Roblox, sesuai cakupan (scope) yang kamu setujui, aplikasi ini menerima dari Roblox:

  • openid, profile — User ID, username, dan nama tampilan akun kamu.
  • asset:read / asset:write — izin melihat & mengunggah aset ke akun/grup kamu.
  • universe-place:write — izin mem-publish versi Place pada experience yang kamu tentukan sendiri.
  • group:read — daftar grup Roblox tempat kamu punya peran Owner/Admin/Developer, dipakai untuk mengisi pilihan "upload sebagai Group".

Cara kami menyimpan data

  • Access token & refresh token dari Roblox disimpan di sesi server (cookie terenkripsi, httpOnly), tidak pernah dikirim ke browser dalam bentuk terbaca atau disimpan di localStorage.
  • Kami tidak pernah meminta atau menyimpan password Roblox kamu maupun cookie sesi (.ROBLOSECURITY) — semua login terjadi langsung di halaman resmi roblox.com.
  • Preferensi non-sensitif (misal User ID/Group ID target upload terakhir, jenis aset terakhir dipilih) disimpan di localStorage browser kamu sendiri, tidak dikirim ke server manapun selain saat kamu melakukan aksi upload.
  • File yang kamu upload (.rbxm/.rbxmx/.mp3/.ogg/.rbxl/.rbxlx) diteruskan langsung ke Roblox Open Cloud API dan tidak disimpan permanen di server kami.

Cara mencabut akses

Kamu bisa mencabut akses aplikasi ini kapan saja lewat:

  • Tombol Logout di aplikasi ini, atau
  • Halaman Roblox: Settings → Security → Manage Authorized Third-Party Apps, cari nama aplikasi ini lalu klik Revoke.

Kontak

Untuk pertanyaan terkait privasi, hubungi pengembang aplikasi ini melalui akun Roblox pemilik aplikasi.
